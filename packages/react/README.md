# @instrumentsjs/react — reserved, deferred until demanded

Deliberately empty (architecture doc: Deferred Until Demanded). The react-three-fiber lesson says a first-class React wrapper hugely expands reach — but shipping it before the vanilla API stabilizes doubles churn. Revisit ~Q3 after v0.1: customer zero (a React app) consumes `instruments.js` core directly until then, which stress-tests the framework-free DX story.
